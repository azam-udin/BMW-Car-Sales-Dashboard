# BMW-Car-Sales-Dashboard
Interactive Excel dashboard analyzing BMW car sales data with KPI cards, pivot tables, slicers, and charts.
This Excel dashboard analyzes 50,000 BMW car sales records to uncover pricing trends, classification performance, and regional fuel preferences. It combines KPI cards, pivot tables, slicers, and charts to deliver a dynamic, interactive view of sales behavior.
